# 🌍 Global Economic Indicators

A comprehensive data analysis project exploring global economic trends using descriptive statistics, regression analytics, and data visualizations.

## 📌 Overview

This project analyzes global economic indicators across multiple countries and time periods, uncovering patterns and relationships through statistical methods and visual storytelling.

## 🔍 What's Inside

- **Descriptive Statistics** — Summary statistics across key economic indicators including GDP, inflation, unemployment, and trade balance
- **Regression Analysis** — Linear regression models to identify relationships between economic variables
- **Visualizations** — Charts and plots built with Matplotlib and Seaborn to communicate findings clearly

## 🛠️ Tools & Libraries

| Tool | Purpose |
|------|---------|
| Python | Core analysis language |
| Pandas | Data manipulation and cleaning |
| NumPy | Numerical computation |
| Matplotlib | Data visualization |
| Seaborn | Statistical plotting |
| Scikit-learn | Regression modeling |

## 📁 Project Structure

```
global-economic-indicators/
│
├── data/               # Raw and cleaned CSV datasets
├── notebooks/          # Jupyter notebooks with analysis
├── visuals/            # Exported charts and plots

```

## 📊 Key Findings

- Identified strong correlations between GDP growth and employment rates across G20 nations
- Regression model achieved significant predictive accuracy on inflation trends
- Visualizations reveal regional economic disparities over the past two decades
